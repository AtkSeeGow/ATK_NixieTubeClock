Python

NixieTubeClock.Python

這是一個單純的小程式，每秒透過Serial Port將現在的時間傳給Arduino


import serial
import time

from datetime import datetime

ser = serial.Serial("COM4", 9600);

while True:
    message = datetime.now().strftime('%H%M%S') + '\n'
    ser.write(str.encode(message));
    time.sleep(1)
